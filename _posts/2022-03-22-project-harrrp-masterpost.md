Customizable machine-readable harp tabs for single note training.

<!--more-->

### About

This is a personal non-profit project. Workflow goes like this:
1. Stumble upon a neat and catchy melody
2. Notate it by ear or with the help of open and appropriately credited sources
3. Adjust the result for the standard harmonica
4. Notate it in an unambiguous and machine-readable manner
5. Play it without the need to memorize it

*Full disclosure: provided content was made without any proper musical education or insight.*

### Scope

The project constitutes of two parts:
- Machine-readable tabs
- ~~Complimentary software~~ *work in progress, please see below*

Raw unrendered tabs can be easily viewed &mdash; they are human-friendly `yaml` files with lots of metadata and whitespaces.

Complimentary software contains several functions which:
- Render existing tabs to `txt` ~~and `pdf` formats~~
- Shift song pitch and mark unavailable notes depending on the harp type
- Switch to custom notation symbols
- Estimate song difficulty and maintain an index table with all calculated parameters
